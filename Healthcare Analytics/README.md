# Healthcare Cost Analytics (Production Grade)

## 🧩 Problem

Analyze healthcare costs and identify key drivers of high medical expenses.

---

## 🔄 Pipeline

Kaggle Data → SQL Staging → Cleaning → Validation → Feature Engineering → Python Analysis → ML Modeling

---

## 📊 Key Analysis

* Cost distribution
* Impact of smoking on charges
* BMI vs cost relationship
* Risk scoring

---

## 📈 Key Insights

* Smokers incur significantly higher costs
* BMI strongly correlates with charges
* Risk score helps identify high-cost patients

---

## 🧪 Statistical Findings

* T-test confirms significant difference in costs between smokers and non-smokers

---

## 🤖 Model

* Linear Regression used
* R² ≈ 0.78

---

## ⚠️ Assumptions

* Invalid BMI removed
* Negative charges filtered
* Categorical normalization applied

---

## 🛠️ Tools

* SQL (PostgreSQL)
* Python (Pandas, NumPy, SciPy, Scikit-learn, Seaborn)

---

## 🚀 How to Run

1. Load raw data
2. Run SQL pipeline
3. Export cleaned data
4. Run Python scripts
5. Review outputs
