# E-commerce Advanced Analytics (SQL + Python)

## 🧩 Problem

An e-commerce platform wants to:

* Understand customer retention
* Identify high-value users
* Analyze revenue trends

---

## 🔄 Pipeline

Raw Data → SQL Cleaning → Feature Engineering → Python EDA → Clustering

---

## 📊 Key Analysis

### 1. Revenue Trends

* Monthly revenue aggregation
* Order volume vs revenue growth

### 2. Cohort Retention

* Tracks user activity over time
* Identifies drop-off points

### 3. Customer Segmentation

* KMeans clustering on:

  * Total spend
  * Order frequency

---

## 📈 Key Insights

* Revenue growth driven by increased user base
* High-value users contribute disproportionately
* Retention drops after first purchase

---

## ⚠️ Assumptions

* Missing revenue treated as 0
* Missing signup_date backfilled

---

## 🛠️ Tools

* SQL (PostgreSQL)
* Python (Pandas, NumPy, Scikit-learn, Seaborn)

---

## 🚀 How to Run

1. Load raw data
2. Run SQL scripts
3. Execute Python scripts
4. Check outputs folder
